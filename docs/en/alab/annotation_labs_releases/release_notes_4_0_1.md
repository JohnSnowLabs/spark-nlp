## 4.0.1

Release date: **16-09-2022**

Annotation Lab 4.0.1 offers navigation buttons in Project Teams Page and Product Stabilization. 
Here are the highlights of this release:

### Highlights
- Navigation buttons on the Project Team page. In version 4.0.1, two navigation buttons Back and Next were added to the Team page. The Back button navigates to the Project's Details page and the Next button to navigates to the Configuration page.
- Disable the Import button while importing large files. When importing big files (e.g. as it is often necessary in the case of Visual NER projects) the import process is not instant. In those cases, the user should have a visual indication that the task is still in progress. As such, the Import button is now disabled immediately after import starts.
- Navigate to the Project dashboard using breadcrumb. It is now possible to navigate to the Project page by clicking on the breadcrumb link
