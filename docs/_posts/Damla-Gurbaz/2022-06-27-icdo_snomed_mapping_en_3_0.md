## How to use



<div class="tabs-box" markdown="1">

```python
from sparknlp.pretrained import PretrainedPipeline

pipeline= PretrainedPipeline("icdo_snomed_mapping", "en", "clinical/models")

result= pipeline.fullAnnotate("8120/1 8170/3 8380/3")
```
