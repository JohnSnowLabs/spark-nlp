## How to use



<div class="tabs-box" markdown="1">
{% include programmingLanguageSelectScalaPythonNLU.html %}
```python
cleaner = ImageTextCleaner \
            .pretrained("text_cleaner_v1", "en", "clinical/ocr") \
            .setInputCol("image") \
            .setOutputCol("cleaned_image") \
            .setMedianBlur(0) \
            .setSizeThreshold(1) \
            .setTextThreshold(0.3) \
            .setPadding(2) \
            .setBinarize(False)
```
