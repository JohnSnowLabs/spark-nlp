## Description

This Text Classifier will identify whether a broker's report suggests to Buy,Sell, Hold, Accumulate, Reduce, Neutral, Other.

## Predicted Entities

`Buy`, `Sell`, `Hold`, `Accumulate`, `Reduce`, `Neutral`, `Other`
