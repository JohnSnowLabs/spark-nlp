Download a pre-trained DocMapperModel.

Args:

name (str): Name of the pre-trained model. lang (str): Language of the pre-trained model. remote_loc (str): Remote location of the pre-trained model. If None, use the open-source location. Other values are “clinical/models”, “finance/models”, or “legal/models”.

Returns:
