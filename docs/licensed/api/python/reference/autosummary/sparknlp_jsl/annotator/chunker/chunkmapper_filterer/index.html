Filters chunks mapped or not by ChunkMapper.

Filters chunks based on whether the ChunkMapper annotator successfully mapped the chunk or not.

Input Annotation types

Output Annotation type

CHUNK,``LABEL_DEPENDENCY``

CHUNK
Parameters:
returnCriteria (str)

Has two possible values: “success” or “fail”. If “fail” (default), returns the chunks that are not in the label dependencies; if “success”, returns the labels that were successfully mapped by the ChunkMapper annotator.
