calculate_proportion(f1_score: float) → [float, None][source]#

This function calculates the proportion value for robustness based on F1 score. It assumes that noise_prob assigned during testing is set to the default value of 0.5. :param f1_score: f1_score of robustness testing results. :return: Corresponding proportion value for f1_score.

suggest_perturbations(test_results: dict) → dict[source]#

This function calculates proportion value for each entity and each test types for robustness fixing. Calculation based on robustness test results and output format should be dictionary. :param test_results: Test result’s metric dictionary obtained from robustness test. :return: Dictionary of suggestions containing test name, entity and proportion.

get_augmentation_proportions(suggestions: dict, key: str) → dict[source]#

This function returns the augmentation proportions for each test. :param suggestions: Dictionary of suggestions containing test name, entity and proportion. :param key: The dictionary key to pull from the suggestions dictionary.
