conll_reader(conll_path: str) → List[tuple][source]#

Read CoNLL file and convert it to the list of labels and sentences. :param conll_path: CoNLL file path. :return: data and labels which have sentences and labels joined with the single space.

conll_writer(sentences: List[str], pos_tags: List[str], chunk_tags: List[str], labels: List[str], save_path: str, docs_indx: Optional[List[int]] = None) → int[source]#

Takes sentences_list, tags_list and labels_list and write to CoNLL file, docs can be separated with docs_indx. :param sentences: List of sentences, tokens are joined with whitespace. :param pos_tags: List of pos tags. :param chunk_tags: List of chunk tags. :param labels: List of labels. :param docs_indx: List of integers that keep track of documents in CoNLL file. :param save_path: CoNLL file path to save. :return: number of sentences written to the file.

update_with_model_predictions(conll_path: str, fix_df: pandas.DataFrame, threshold: float = 0.3, save_path: Optional[str] = None) → str[source]#

A method to update CoNLL with label fixes. Takes output dataframe of test_label_errors as input. :param conll_path: Path to CoNLL data which will be fixed with replacements. :param fix_df: test_label_errors output, dataframe consist of sent_indx, token_indx and correct label. :param threshold: minimum error score for replacements. :param save_path: Path to CoNLL data to save new data with replacements. If None, new file will be created with a new name automatically. :return: info string with the name of output file and total number of changes.

apply_label_fixes(conll_path: str, label_fixes: Dict[Tuple[int, int], str], save_path: str = None) → int[source]#

A method to update CoNLL with label fixes. Takes CoNLL data path and dictionary of label fixes as input. :param conll_path: Path to CoNLL data which will be fixed with replacements. :param label_fixes: Dictionary where sentence and token indexes are mapped to replacements. Keys should be passed in tuple format, e.g (sent_index, token index) and values should be corresponding entity to be replaced. :param save_path: Path to CoNLL data to save new data with replacements. If None, new file will be created with a new name automatically. :return: total number of label fixes.

add_flag_to_conll(conll_path: str, flag_indexes: List[Tuple[int, int]], flag: Optional[str] = '*', save_path: Optional[str] = None) → str[source]#

A method to add label flag to CoNLL data. Takes CoNLL data path and dictionary of flag fixes as input. :param conll_path: Path to CoNLL data. :param flag_indexes: List of tuples where each tuple have sent_indx and token_indx, e.g (sent_indx, token_indx) :param flag: String that will be replaced with part of speech and syntactic tag in the CoNLL file. Default is *. :param save_path: Path to CoNLL data to save new data. If None, new file will be created with a new name automatically. :return: info string with the name of output file and total number of changes.
