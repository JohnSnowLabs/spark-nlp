create_graph(model_location: str, ntags: int, embeddings_dim: int, nchars: int, lstm_size: int = 128, model_filename: str = None, gpu_device: int = 0, is_medical: bool = False)[source]#

Creates the Tensorflow graph for NERDL model

Args:

model_location (str): Folder where the model will be saved ntags (int): Number of unique tags embeddings_dim (int): Dimension of the word embeddings nchars (int): Number of unique characters lstm_size (int, optional): Size of the hidden LSTM cells. Defaults to 128. model_filename (str, optional): Name of the model when saved on disk. Defaults to None. gpu_device (int, optional): Which GPU to use. Defaults to 0. is_medical (bool, optional): If the model being trained is from Healthcare NLP or not. Defaults to False.
