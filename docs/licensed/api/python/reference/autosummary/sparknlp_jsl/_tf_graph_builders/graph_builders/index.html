class GenericClassifierTFGraphBuilder(build_params)[source]#

Class to create the the TF graphs for GenericClassifierApproach. Extends the TFGraphBuilder base class.

Examples

>>> from sparknlp_jsl.training import tf_graph
>>> from sparknlp_jsl.base import *
>>> from sparknlp.annotator import *
>>> from sparknlp_jsl.annotator import *
>>> from sparknlp_jsl.annotator import *
>>> dataframe = pd.read_csv('petfinder-mini.csv')
>>> DL_params = {
...     "input_dim": 302,
...     "output_dim": 2,
...     "hidden_layers": [300, 200, 100],
...     "hidden_act": "tanh",
...     "hidden_act_l2": 1,
...     "batch_norm": 1,
...     "output_act": "softmax",
...     "loss_func": "cross_entropy",
... }
...
>>> tf_graph.build("generic_classifier", build_params=DL_params, model_location="/content/gc_graph", model_filename="auto")
This created the graph and saved it to /content/gc_graph/gcl.302.2.pb
We can use it in the GenericClassifierApproach annotator by setting the modelFile parameter
>>> gen_clf = GenericClassifierApproach() \
...    .setLabelColumn("target") \
...    .setInputCols(["features"]) \
...    .setOutputCol("prediction") \
...    .setModelFile('/content/gc_graph/gcl.302.2.pb') \
...    .setEpochsNumber(50) \
...    .setBatchSize(100) \
...    .setFeatureScaling("zscore") \
...    .setFixImbalance(True) \
...    .setLearningRate(0.001) \
...    .setOutputLogsPath("logs") \
...    .setValidationSplit(0.2)

>>> clf_Pipeline = Pipeline(stages=[features_asm,gen_clf])

static build(model_name, build_params, model_location, model_filename='auto')[source]#

Method that create the tf graph.

Parameters:
model_name: str

The name of the tf model that you want to build. Available models are ner_dl, generic_classifier, assertion_dl and relation_extraction.

build_params: dict

Configuration params to build the tf graph for the specific model.

model_location: str

Path where the model will be saved.

model_filename: str

Name of the .rb file. If “auto” (default), the filename will be generated.

Raises:
Exception

If the model_name is not supported. If model_filename is “auto” but model_name doesn’t support automatic filename generation.

ModuleNotFoundError

If tensorflow 1.15 or 2.xx is not installed.

WrongTFVersion

If the tf version is not 1.15 or 2.xx.

TensorflowAddonsNeeded

If tensorflow addons is not installed.

Examples

>>> from sparknlp_jsl.training import tf_graph
>>> tf_graph.build("assertion_dl",build_params={"n_classes": 10}, model_location="/tmp", model_filename="assertion_dl.pb")
